Welcome to the Protein API

This API allows you to access information about proteins and their domains.

Endpoints:

  • POST https://bio-science-research-eb2589b82ae4.herokuapp.com/api/protein/ - add a new record
  • GET https://bio-science-research-eb2589b82ae4.herokuapp.com/api/protein/[PROTEIN ID] - return the protein sequence and all we know about it
  • GET https://bio-science-research-eb2589b82ae4.herokuapp.com/api/pfam/[PFAM ID] - return the domain and it's description
  • GET https://bio-science-research-eb2589b82ae4.herokuapp.com/api/proteins/[TAXA ID] - return a list of all proteins for a given organism
  • GET https://bio-science-research-eb2589b82ae4.herokuapp.com/api/pfams/[TAXA ID] - return a list of all domains in all the proteins for a given organism
  • GET https://bio-science-research-eb2589b82ae4.herokuapp.com/api/coverage/[PROTEIN ID] - return the domain coverage for a given protein
